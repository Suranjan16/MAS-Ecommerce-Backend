# MAS Ecommerce Backend

Backend API for the MAS Ecommerce application built using Spring Boot.
## Author

Suranjan Maity

## Features

- JWT Authentication & Authorization
- Role-based access (USER / ADMIN)
- Product Management
- Cart Management
- Order Placement
- Order History
- Order Cancellation
- Razorpay Payment Integration
- Cash on Delivery (COD)
- Payment Verification
- Secure REST APIs

## Tech Stack

- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- MySQL
- Maven
- Razorpay API

## API Modules

### Authentication
- Signup
- Login
- JWT Token generation

### Products
- Add product
- Update product
- Delete product
- Get all products

### Cart
- Add to cart
- Remove item
- Update quantity
- Clear cart

### Orders
- Place order
- Get order history
- Get order by id
- Cancel order
- Update order status

### Payments
- Razorpay payment creation
- Payment verification
- COD support
